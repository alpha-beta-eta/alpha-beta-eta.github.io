Lem编辑器使用笔记

Lem编辑器是一个类Emacs编辑器, 但是使用Common Lisp作为其扩展语言.

默认情况下的常用快捷键

  • M-x: 执行命令
  • Tab: 补全 (例如输入路径或者命令时)
  • C-x C-s: 保存当前缓冲区
  • C-x C-f: 打开文件
  • C-x C-w: 写入文件
  • C-+: 增大字号
  • C--: 缩小字号
  • C-s: 搜索
  • M-g: 行号跳转
  • M-<: 跳转至缓冲区开头
  • M->: 跳转至缓冲区末尾
  • M-w: 复制
  • C-w: 剪贴
  • C-y: 粘贴
  • C-x k: 关闭缓冲区
  • C-x b: 切换/新建缓冲区
  • C-x C-b: 列出所有缓冲区 (并可选择切换)
  • C-x Left: 切换至前一个缓冲区
  • C-x Right: 切换至后一个缓冲区
  • C-x 2: 上下分屏
  • C-x 3: 左右分屏
  • C-x 1: 删除其他所有窗口
  • C-x 0: 删除当前窗口
  • M-:: 对于Common Lisp表达式求值
  • C-x C-c: 退出Lem编辑器

默认情况下的常用命令

  • find-file: 打开文件
  • write-file: 写入文件
  • start-lisp-repl: 打开Common Lisp的REPL
  • toggle-line-numbers: 打开/关闭行号
  • toggle-tabbar: 打开/关闭缓冲区标签栏 (似乎存在bug)
  • list-buffers: 列出所有缓冲区 (并可选择切换)
  • list-modes: 列出所有模式
  • lisp-mode: 切换至Common Lisp模式
  • scheme-mode: 切换至Scheme模式
  • list-color-themes: 列出所有主题 (并可选择切换)
  • load-theme: 加载主题
  • describe-bindings: 描述当前的快捷键绑定

配置

  • define-color-theme: 定义主题
  • define-command: 定义命令
  • define-key: 定义快捷键
