第1章 分布函数

第1.1节 单调函数

本书的正负增减均为广义, 即x为正指的是x≥0. 若要使用传统意义, 则须加严格二字.

设f:ℝ→ℝ是一个增函数, 则对于任意两个实数x1和x2, 我们有 (1) x1 < x2 ⇒ f⁡ ( x1 ) ≤ f⁡ ( x2 ) . (1) 当然了, 这也等价于 x1 ≤ x2 ⇒ f⁡ ( x1 ) ≤ f⁡ ( x2 ) .

我们使用t↑x表示t从左侧趋近于x, t↓x表示t从右侧趋近于x.

命题1.1.1. 对于每个x, f在x处的左极限和右极限均(严格)存在, 其记号分别为 (2) f⁡ ( x− ) = lim t↑x f⁡ (t) 和 f⁡ ( x+ ) = lim t↓x f⁡ (t) . (2) f在无穷远处的极限亦(广义)存在, 其记号分别为 f⁡ ( −∞ ) = lim t↓ −∞ f⁡ (t) 和 f⁡ ( +∞ ) = lim t↑ +∞ f⁡ (t) . 前者可能为−∞, 后者可能为+∞.
证明. 对于任意的实数x, 设 a= sup t<x f⁡ (t) . 根据单调性和确界原理, 这个上确界是良定的. 对于每个ε>0, 存在δ>0满足 f⁡ ( x−δ ) > a−ε . 根据单调性, 对于每个实数t满足x−δ<t<x, 我们有 f⁡ (t) > a−ε . 并且, 我们还知道 f⁡ (t) ≤a . 因此, 可以推出 | f⁡ (t) −a | <ε . 这告诉我们f在x处的左极限存在, 且 f⁡ ( x− ) = sup t<x f⁡ (t) . 如法炮制, 我们可以证明 f⁡ ( x+ ) = inf t>x f⁡ (t) . 对于趋向无穷远处的极限, 证明也是类似的.
◻
命题1.1.2. 对于每个实数x, f在x处连续当且仅当 f⁡ ( x− ) = f⁡ (x) = f⁡ ( x+ ) .
证明. 函数f在x处连续当且仅当 lim t→x f⁡ (t) = f⁡ (x) . 而这又当且仅当 f⁡ ( x− ) = f⁡ (x) 且 f⁡ ( x+ ) = f⁡ (x) . 另外, 原书的证明方法给出了 (3) f⁡ ( x− ) ≤ f⁡ (x) ≤ f⁡ ( x+ ) (3) 作为中间步骤.
◻
译者注记. 这个命题对于一般的函数也成立.

一般而言, 我们称f在x处有一个跳跃, 如果(2)中的两个极限均存在且不相等. f在x处的值本身, 即f⁡(x), 则可以是任意的. 但对于增函数f而言, 关系(3)必然成立. 作为命题1.1.1和命题1.1.2的推论, 我们有以下结果.

命题1.1.3. 一个增函数的唯一可能的非连续性种类为跳跃. {读者还应该自问, 对于一般的函数, 还存在哪些种类的非连续性.}

如果x处存在跳跃, 那么我们称x是f的一个跳跃点, 而数字f⁡(x+)−f⁡(x−)被称为f在x处跳跃的大小.

注意跳跃点的集合不必是一个闭集.

例子1.1.1. 令x0是一个任意的实数, 并定义函数f如下: f⁡ (x) = { 0 , 对于 x< x0 −1 1− 1n , 对于 x0 − 1n ≤x< x0 − 1 n+1 , 其中 n= 1,2,… 1 , 对于 x≥ x0 点x0是跳跃点集合{x0−1/n|n≥2}的一个聚点, 但是f在x0处连续.

例子1.1.2. 令(an)n≥1是任意给定的对于有理数集全体的枚举, 令(bn)n≥1是正序列 (即非负序列) 满足∑n=1∞bn<∞. 例如, 我们可以取bn=2−n. 现在考虑 (5) f⁡ (x) = ∑ n=1 ∞ bn ⁢ δ an ⁡ (x) . (5) 既然对于每个n和x都有0≤δan⁡(x)≤1, 那么(5)中的级数是绝对和一致收敛的. 鉴于每个δan都是增函数, 可以推出如果x1<x2, 那么 f⁡ ( x2 ) − f⁡ ( x1 ) = ∑ n=1 ∞ bn ⁢ [ δ an ⁡ ( x2 ) − δ an ⁡ ( x1 ) ] ≥0 . 因此, f是递增的. 感谢一致收敛, 我们可以推出对于每个x, 都有 (6) f⁡ ( x+ ) − f⁡ ( x− ) = lim t↓x ∑ n=1 ∞ bn ⁢ δ an ⁡ (x) − lim t↑x ∑ n=1 ∞ bn ⁢ δ an ⁡ (x) = ∑ n=1 ∞ lim t↓x bn ⁢ δ an ⁡ (x) − ∑ n=1 ∞ lim t↑x bn ⁢ δ an ⁡ (x) = ∑ n=1 ∞ bn ⁢ δ an ⁡ ( x+ ) − ∑ n=1 ∞ bn ⁢ δ an ⁡ ( x− ) = ∑ n=1 ∞ bn ⁢ [ δ an ⁡ ( x+ ) − δ an ⁡ ( x− ) ] (6) 但是对于每个n, 以上方括号里的数字要么是0要么是1, 根据x≠an还是x=an. 因此, 如果x不同于所有的an, 那么(6)的右侧的每个项就都归零了; 另一方面, 如果x=ak, 那么右侧恰存在一个项没有归零, 也就是n=k的情形, 那么这整个级数产生的值就是bk. 这证明了函数f在所有的有理点处均有跳跃, 而在其他地方均无跳跃. {译注: 当然如果要这样, (bn)n≥1的选取就得保证每一项是严格正的了.}

这个例子表明了跳跃点集可能处处稠密; 实际上这个例子里的有理数集可以改成任意的可数集合而并不改变论证方式. 现在我们要表明可数性的条件是不可避免的. 可数总是指有限或可数无限. {译注: 不过, 在这个例子里, 可数仅能是可数无限, 否则的话并不存在枚举. 然而, 在有限的情况下, 我们也可以对于论证进行少许修改.}

命题1.1.4. f的不连续点集合是可数的.
证明. 我们将使用具有一般适用性的拓扑方法进行论证. 本节后的练习3将指出基于同样有用的计数方法的另外一种证明. 对于每个跳跃点x, 我们考虑开区间Ix=(f⁡(x−),f⁡(x+)). 如果x′是另一个跳跃点, 比如说x<x′, 那么存在一个点x∼满足x<x∼<x′. 于是, 根据单调性, 我们有 f⁡ ( x+ ) ≤ f⁡ ( x∼ ) ≤ f⁡ ( x′ − ) . 因此, 两个区间Ix和Ix′互不相交, 虽然如果f⁡(x+)=f⁡(x′−)则会邻接. 于是, f的定义域中的跳跃点的集合可与f的陪域中的特定的互不相交的开区间类联系起来. 那么, 任何这样的类都是可数的, 因为每个区间都包含一个有理数, 于是这区间类能够一一对应于有理数集的某个特定子集, 而后者必然是可数的. 因此, 不连续点集合也是可数的, 鉴于其与这开区间类是一一对应的.

证明. 为了证明这一论断, 令x是一个任意的点, 并令tn∈D, tn′∈D, tn↑x, tn′↓x. 鉴于D稠密, 故这样的序列存在.
